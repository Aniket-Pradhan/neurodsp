Neuro Digital Signal Processing (neurodsp)
==========================================

neurodsp is a collection of python modules routinely used for analysis of time series in neuroscience, with a focus on time-frequency and spectral analysis, as well as simulation.

If you're just getting started, you might want to start with the Readme_, which also contains installation instructions.

If you want to explore how everything works, check out the Tutorial_.
